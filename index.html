Protótipo de Jogo em C++ com biblioteca SDL 🕹️

Site “PokeTrader” com uso de API, PHP e banco de dados MySQL ✨

Este site foi desenvolvido no curso “Técnico em Informática” durante a disciplina de “Programação WEB”. Neste projeto, foram usadas as 3 linguagens padrão para sites: HTML, CSS e Java Script, além do uso de PHP em conjunto com um banco de dados MySQL para tornar o conteúdo do site dinâmico. O objetivo do site é servir como um trocador de Pokémons, em que um usuário (com conta criada) recebe um pokémon a cada dia que entra no site, além de poder trocar os seus pokémons com o de outro usuário. Dessa forma, atraindo a atenção do usuário que pode colecionar os seus monstros e buscar outros usuários para trocar. (2022)

Link pro projeto: Projeto PokéTrader

Monitor em Curso de Arduino ⚡

Em 2022 fui bolsista de um projeto que tinha como objetivo ministrar um curso de Arduino, microcontrolador de circuitos eletrônicos. Este curso foi ofertado aos alunos de Informática do IFPR Paranavaí, e incluiu de minha parte preparar o material didático, realizar apresentações orais-expositivas e corrigir as atividades propostas. O curso consistiu em torno de 10 aulas, que ocorreram à tarde, das 13:30 - 17:00.

Tutor de Informática no Instituto Maurício Gehlen 👵📲

Em 2025, até o presente momento, faço um serviço de tutoria de informática para os alunos, da terceira idade, do Instituto Maurício Gehlen em Paranavaí. A atividade consiste em auxiliar alunos com eventuais dúvidas e dificuldades com o celular, seja em relação ao conhecimento de como operar (salvar contato, fazer cadastro em um aplicativo, baixar um jogo, excluir aplicativos), ou relativo à problemas técnicos com os dispositivos (celular lento, travando, aparecendo anúncios, memória cheia). A minha participação nesse projeto tem a duração prevista de 1 mês, e acontece de manhã, das 08:00 às 11:30.
